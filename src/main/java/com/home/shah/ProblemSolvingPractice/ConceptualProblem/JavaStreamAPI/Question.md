# 🌊 Java Stream API Practice Questions

Master the Java Stream API with these progressive challenges, from beginner to expert level.

---

## 🎯 Beginner Level

*Getting started with basic stream operations*

1. **Convert to Uppercase** 📝  
   Convert a list of strings to uppercase using streams.

2. **Filter Even Numbers** 🔢  
   Filter even numbers from a list of integers.

3. **Count Strings Starting with "A"** 🔤  
   Count how many strings in a list start with the letter "A".

4. **Sort in Descending Order** ⬇️  
   Sort a list of integers in descending order using streams.

5. **Remove Duplicates** 🗑️  
   Remove duplicate elements from a list using streams.

---

## 🚀 Intermediate Level

*Chaining operations and using collectors*

6. **Group by Department** 👥  
   Group a list of employees by department using `Collectors.groupingBy`.

7. **Find Longest String** 📏  
   Find the longest string in a list using `reduce`.

8. **Comma-Separated String** ✂️  
   Convert a list of strings to a comma-separated string using `Collectors.joining`.

9. **Partition Even/Odd** ⚖️  
   Partition a list of integers into even and odd using `Collectors.partitioningBy`.

10. **Second-Highest Number** 🥈  
    Find the second-highest number in a list using streams.

---

## 💡 Advanced Level

*Deep understanding of stream mechanics and custom collectors*

11. **Flatten Lists** 🔄  
    Flatten a list of lists of integers into a single list using `flatMap`.

12. **Word Frequency Counter** 📊  
    Implement a custom collector to count word frequency in a list of strings.

13. **Debug with peek()** 🔍  
    Use `peek()` to debug a stream pipeline and explain its behavior.

14. **Complex Object Graph** 🗺️  
    Write a stream pipeline that filters, maps, and collects data from a complex object graph  
    *(e.g., `List<Order>` → `List<ProductName>`)*.

15. **Parallel vs Sequential** ⚡  
    Parallelize a stream operation and measure performance difference with sequential stream.

---

## 🏆 Expert Challenges

*Push the boundaries of stream usage and performance*

16. **Paginated Sublist Utility** 📄  
    Build a reusable utility method that transforms any `List<T>` into a paginated sublist using streams.

17. **Frequency Histogram** 📈  
    Write a stream-based implementation of a frequency histogram from a large text file.

18. **Graceful Exception Handling** ⚠️  
    Create a stream pipeline that handles checked exceptions gracefully  
    *(e.g., I/O operations inside map)*.

19. **Multi-Field Deduplication** 🔐  
    Implement a stream-based deduplication logic for a list of objects based on multiple fields.

20. **ETL Pipeline** 🔄📊  
    Design a stream-based ETL pipeline that reads from a CSV, transforms data, and writes to another format.
